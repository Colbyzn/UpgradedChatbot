# ChatRobot

#### Description
该项目为简易版的聊天机器人，原理是使用 fetch 请求一个 API 接口，并把数据展示出来。这个 API 是某个开发团队提供的免费接口，用来做智能回答的。
利用这个神奇的 API ，我们就可以简单地实现一个功能，就是在输入框输入你想说的话，接着点击发送，然后 请求 API 接口，最后把回复展示出来，一个聊天机器人就做成了~
